# 東京ディズニーリゾート マップクイズ

個人用の非公式ファンメイドWebアプリです。公式ロゴ、公式地図、キャラクター画像は使わず、自作の簡易マップUIで番号ピンから施設名を当てます。

## 起動

```powershell
npm install
npm run dev
```

表示されたローカルURLをブラウザで開いてください。

## 主な機能

- 東京ディズニーランド、東京ディズニーシー、両パーク混合のモード切り替え
- アトラクションのみ、全施設、ファンタジースプリングスのみ、難問モード
- 10問、20問、50問、全問から出題数を選択
- 自作簡易マップ上の番号ピンをクリックして回答
- 選択肢検索、回答変更、localStorageによる途中保存
- 採点結果とSNS向けコピー文の生成

## データ追加

施設データは `src/data/spots.ts` の `spots` 配列で管理しています。

```ts
{
  id: 'tdl_001',
  park: 'land',
  area: 'トゥモローランド',
  category: 'attraction',
  name: 'スペース・マウンテン',
  x: 72,
  y: 35,
  difficulty: 1,
}
```

`x` と `y` は簡易マップ上の相対座標 0〜100 です。
